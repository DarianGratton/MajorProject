### Controls
Player Controls
```
W       - Navigate/Move Up
S       - Navigate/Move Down
A       - Navigate/Move Left
D       - Navigate/Move Right
Enter   - Confirm
K       - Use Weapon 1
L       - Use Weapon 2
Esc     - Pause/Back
```

Enemy Controls
```
Up Arrow     - Navigate/Move Up
Down Arrow   - Navigate/Move Down
Left Arrow   - Navigate/Move Left
Right Arrow  - Navigate/Move Right
O            - Use Weapon 1
P            - Use Weapon 2
```
